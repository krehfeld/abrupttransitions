This repository contains data and materials to investigate how abrupt changes in the Earth System can be predicted based on observations and statistics.

The *references* folder contains relevant literature, the *data* folder contains the data for exercises.

Materials included for the *tutorials* are under CC-BY 4.0 license (https://creativecommons.org/licenses/by/4.0/deed.en)
